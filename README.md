# URL to fetch and Filter 'uFilter'

#Install & Set :
```
> git clone https://github.com/RootKRD/Filter
> cd Filter
> ruby uFilter.rb
```

# Usage : 

```
ruby uFilter.rb -h
ruby uFilter.rb https://example.com -txt
ruby uFilter.rb https://example.com -txt -js -json
```
